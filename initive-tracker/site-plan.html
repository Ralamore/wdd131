Overview

Purpose

To track initive for dungeon masters playing Dungeons and Dragons 5th Edition, including order and conditional effects.

Audience

Dungeon Masters for D&D 5th Edition.

Dynamic elements

  • Input into a form records the input and formats it into a block of imported HTML
  • Dropdown menus with options.
  • Moving HTML blocks within the document.

Content

Home page

Logo, welcome message, example player block, a form to fill out with player information (number, pp, ac, ss dc), and a form to fill out similar information for the monsters.

Initive Tracker

Player blocks in a column along the left side, each with an initive space; a button to start the combat; a round count in the top right corner; and a section along the right to display further info

Wireframes

Home

There will be two forms as you scroll down: one for the player characters and monsters. These will auto generate duplicate blank forms as the user presses a button to add more at the bottom of the form.)

Initive Tracker

The nav bar will have a return to home anchor. The user will input the initive for each player and then click the "Start Combat" button. This will shuffle the player and monster blocks into their initive order. Each time the user presses the "Progress" button, the next block in line will be highlighted. If the block is marked with the "Dead" condition, the block is removed.
